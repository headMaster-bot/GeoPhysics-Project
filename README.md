# GeoPhysics Backend

This is the backend API for the GeoPhysics Project Management System built with Node.js, Express, and MongoDB.

## Features

- JWT Authentication
- User management
- Project management APIs
- Epic and story APIs
- Survey management
- RESTful API architecture
- MongoDB database integration

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
